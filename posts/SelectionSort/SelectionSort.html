Introduction

Sorting a dataset involves arranging the data collection to establish an accurate relationship, in this case, order. Sorting is one of the most studied problems in computer science, resulting in many algorithms of varying efficiency that sort elements such as names, indices, letters, etc. (Necaise 2011). Selection sort is one of these sorting algorithms and is based on creating a new list where it places the values it is sorting. Its efficiency is \[ O(n^{2}) \] .

Recursion is a process for solving problems and a powerful programming tool to solve a larger problem by subdividing it into smaller cases, then solving the smaller problem to provide a solution to the initial problem. It can be used with a wide range of problems, from basic traditional iterations to more advanced backtracking problems. In some cases, recursion is not the efficient and elegant solution (Bhargava 2024).

Exercise 1

A. Problem Statement

Implement the Selection Sort algorithm.

The Selection Sort algorithm consists of two functions. The first function, called findSmallest, is responsible for returning the index of the smallest element in the array. The second function is responsible for performing the sorting by placing the element from the index returned by the findSmallest function into a new array and removing this element from the original array. This process is repeated iteratively until all elements in the array are consumed.

def findSmallest(arreglo):
    """
    funcion  que retorna el incice del valor mas pequeno del arregloeglo o lista
    Inpust:
        arreglo: arregloeglo de entrada
    Returns:
        indice_valor_mpequeno (int): indice del elemnto mas pequeno
    """
    valor_mpequeno = arreglo[0] #tomamos el primer elemnto dl arreglo 
    indice_valor_mpequeno = 0 #se asgina indice 0
    for i in range(1, len(arreglo)): #itermos para ir comparando
        if arreglo[i] < valor_mpequeno: #realizamos la comparacion elemnto a elemento
            valor_mpequeno = arreglo[i]
            indice_valor_mpequeno = i
    return indice_valor_mpequeno #retornamos indice
def selectionSort(arreglo):
    """
    Algoritmo de ordenamiento retorna el arregloeglo ordenado de menor a mayor
  
    Inpust:
        arregloeglo: arregloeglo o lista a ordenar menor a mayor
    Returns:
        arregloeglo : ordenado 
    """
    nuevo_arreglo = [] #declaramos el arreglo a retornar 
    for i in range(len(arreglo)):#iteramos sobre al arreglo ingresado
        indice_valor_mpqueno = findSmallest(arreglo)#obtenemo indice valor mas pequeno
        nuevo_arreglo.append(arreglo.pop(indice_valor_mpqueno))
    return nuevo_arreglo

A. Problem Statement

Provide an example of recursion.

The implemented recursive function performs the addition of a scalar to a multidimensional array of integers or decimals. The base case of the function checks if it encounters an integer or decimal value; otherwise, it continues calling itself, traversing the entire array. Recursion allows solving this problem as it generalizes the algorithm, which does not happen if we implement it with the same iterative logic, as it only solves the problem for a predefined dimension.

def suma_escalar_array(arreglo, numero):
    """
    funcion que permite sumar un escalar a una lista
    Args:
        arreglo:(list) arreglo al que se va sumar el numero escalar
        numero (int or float) : escalar que va sumar al arreglo
    Returns:
        arreglo (list): arreglo sumado escalar
    """
    if type(arreglo) is  int or type(arreglo) is float:
        return arreglo+numero;
    else :
        for i in range(0,len(arreglo)) :
            arreglo[i]=suma_escalar_array(arreglo[i], numero)
    return arreglo

Conclusions

  • The implementation of selection sort carried out in Exercise 1 confirms the utility of selection sort as a tool for ordering a collection of data.
  • The selection sort algorithm implemented in Exercise 1 is composed of two functions: the first, findSmallest, which returns the index of the smallest element in the array, and the second, selectionSort, which adds this element to the new sorted list and removes it from the original array.
  • The efficiency of selection sort is $ O(n^2) $ operations to sort a list of n elements.
  • The recursive function addScalarToArray implemented in Exercise 2 generalizes the algorithm for adding a scalar to an array of different dimensions. In contrast, implementing the function with the same iterative logic only solves the problem for an array of predefined dimension.
  • Recursion consists of one or more base cases. In the function in Exercise 2, the base case executes when the input value is an integer or decimal, and the recursive case occurs when the input value is a list.
  • The call stack in the function from Exercise 2 will depend on the dimensionality of the elements in the input array.
